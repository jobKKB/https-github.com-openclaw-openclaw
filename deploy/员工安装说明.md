# 舟岱自动化小助手 - 快速安装说明

## 🚀 一分钟快速开始

### 步骤一：解压安装包

将安装包（`zhoudai-install-*.zip`）解压到 `D:\zhoudai\`

### 步骤二：启动

进入 `deploy` 文件夹，**右键 `start.bat` → 以管理员身份运行**

### 步骤三：等待启动

- **有 Docker**：约 1-3 分钟
- **无 Docker（首次）**：约 15-30 分钟（自动安装环境）

### 步骤四：访问

浏览器自动打开 `http://localhost:18788`

---

## 📋 安装要求

| | 推荐 | 最低 |
|--|------|------|
| 系统 | Windows 11 | Windows 10 |
| 内存 | 8GB | 4GB |
| 硬盘 | 10GB | 5GB |

---

## 🔑 获取网关令牌

首次访问需要令牌，在安装目录的 `.env` 文件中查找：
```
ZHOUDAI_GATEWAY_TOKEN=你的令牌在这里
```
